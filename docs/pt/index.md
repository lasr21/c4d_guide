# C4D Guide

Bem-vindo à documentação do C4D Guide!

## Descrição

Este é um site de documentação simples e multilíngue construído com MkDocs e Material for MkDocs.

## Características

- ✨ **Design moderno** com Material Design
- 🌍 **Suporte multilíngue** (Espanhol, Inglês, Português)
- 📱 **Responsivo** - funciona em todos os dispositivos
- 🔍 **Busca integrada**
- 🌙 **Modo escuro/claro**

## Início rápido

Esta documentação foi projetada para ser simples e fácil de navegar. Você pode alterar o idioma usando o seletor de idiomas no canto superior direito.

### Navegação

- Use o menu superior para navegar pelas seções
- O botão de busca o ajudará a encontrar conteúdo específico
- Alterne entre modo claro e escuro conforme sua preferência
